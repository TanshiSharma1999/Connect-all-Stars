# Connect-all-Stars

A fun and interactive reaction-based game built using **Python (Pygame Zero)** where the player must connect all stars in the correct order as fast as possible.

---

## 🎮 Game Overview

In **Connect All Stars**, stars appear at random positions on the screen. The player’s goal is to click each star in the correct sequence (1 → 8).

- Click stars in the correct order
- Lines are drawn between correctly connected stars
- Wrong click resets the game
- Try to finish in the fastest time possible!

---

## 🧠 Features

- ⭐ Random star placement every game
- 🖱️ Simple mouse-based gameplay
- 🔗 Automatic line connections between stars
- ⏱️ Live timer to track performance
- ❌ Reset system for wrong clicks
- 🎨 Background + sprite-based visuals

---

## 🕹️ How to Play

1. Run the game using Pygame Zero
2. Click stars in order from **1 to 8**
3. Avoid mistakes (or you restart!)
4. Complete all stars as fast as possible
